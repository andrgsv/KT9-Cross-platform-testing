
**КТ №9. Кроссплатформенное тестирование (Wikipedia + Appium)**

Описание проекта:
Данный проект демонстрирует кроссплатформенное тестирование веб-приложения Wikipedia
с использованием Selenium (веб) и Appium (мобильное тестирование жестов).

Сайт:
https://www.wikipedia.org/

----------------------------------------------------
1. Web тестирование
----------------------------------------------------
Проверяется:
- корректность отображения сайта Wikipedia на разных ОС (Windows / macOS)
- адаптивность интерфейса (desktop / mobile view)
- работа поиска

Инструменты:
- Selenium WebDriver
- PyTest

----------------------------------------------------
2. Mobile + Appium
----------------------------------------------------
Проверяется:
- работа мобильной версии Wikipedia
- выполнение свайпов (scroll gesture)
- адаптация интерфейса под экран устройства

Инструменты:
- Appium
- Android Emulator
- W3C Actions

----------------------------------------------------
Запуск Web тестов:
----------------------------------------------------
```bash
pip install -r requirements.txt
pytest tests/test_web_wikipedia.py
```

----------------------------------------------------
Запуск Mobile тестов (Appium):
----------------------------------------------------
```text
1. Запустить Appium Server:
   appium

2. Запустить Android Emulator

3. Выполнить тест:
   pytest tests/test_mobile_wikipedia.py
```
